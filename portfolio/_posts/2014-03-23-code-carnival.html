Code Carnival is an educational game to teach children programming fundamentals.

  • Players help fix a robotic carnival using programming.
  • Fundamental concepts of programming are introduced through puzzles.
  • Concepts like zero-based indexing in arrays are the focus of the game.

Project Goal

This project was created for the final project of the Design of Educational Games course at Carnegie-Mellon University. I worked as part of a team of 4 students (2 graduate and 2 undergraduate).

My Contributions

Learning Goals

We wanted to separate the logic of fundamental concepts away from syntax. And we decided to focus on two important concepts: arrays and conditional statements. These learning objectives are tightly coupled into the gameplay. To succeed, the player must learn these concepts in order to play the game. He/she doesnâ€™t play the game with a layer of programming knowledge on top. James Gee found that learning where the student learns everything himself is much more effective.

Study Design

The focus of this game began with a cognitive task analysis of novice and expert programmers.

We then brainstormed a game to teach students programming fundaments focusing on discovery learning.

Paper Prototyping

We playtested the game in three rounds. Two paper prototypes were tested and the final web-based prototype.

We included pre and post tests to analyze the game for learning transfer.

Final Prototype

The final prototype contains three levels. The first and second encourage the player to fix a broken robot and in doing so, they learn to organize information conditionally. The third level is a whack-a-mole game based on an array with zero-based indexing.
